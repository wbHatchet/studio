# **App Name**: AI Lo-Fi Channel Factory

## Core Features:

- AI-Driven Micro-Niche Strategy Tool: Utilizes Claude AI to analyze YouTube trends, competition, and keyword data (e.g., VidIQ) to identify and suggest highly profitable micro-niches, artists, and thematic concepts for new Lo-Fi channels, forming the foundation of the content factory. This tool provides actionable insights for channel strategy.
- Content Project & Channel Management: A user interface allowing creators to define and manage multiple Lo-Fi channel projects, specify content parameters via structured input (e.g., spreadsheet uploads), and oversee the production queue across their channel network. Firestore will store all project, channel, and content metadata.
- AI Music Generation & Curation: Employs Claude AI to craft detailed, niche-specific prompts for AI music generators (e.g., Suno AI), orchestrates the synthesis of instrumental Lo-Fi tracks, and provides tools for curating and arranging a continuous audio playlist (e.g., 30-60 minute loops) from generated beats.
- AI Visuals & Animation Production: Uses Claude AI to generate prompts for AI image tools (e.g., Pika AI, DALL-E) to create custom Lo-Fi artwork and background scenes, then utilizes AI animation tools (e.g., Hailuo AI, Pika AI) to transform these static images into seamless, looping video segments suitable for extended playback.
- Automated Video Assembly & Rendering: Automatically combines AI-generated audio playlists with looping animated visuals and custom overlays (e.g., using FFmpeg or similar tools) to produce high-quality, long-form Lo-Fi videos and vertical YouTube Shorts, ready for publishing.
- AI-Powered YouTube SEO Optimization: A tool that leverages Claude AI to generate comprehensive, micro-niche-optimized YouTube metadata, including engaging titles, detailed descriptions (with call-to-actions, BPM, key, and licensing info), and a strategic set of tags and hashtags, directly aligning with proven monetization strategies.
- Multi-Channel YouTube Publishing & Scheduling: Automates the uploading and scheduling of Lo-Fi content to multiple YouTube channels using the YouTube Data API, supporting diverse formats like long-form videos, Shorts, and persistent 24/7 live streams for maximizing audience reach and watch time. Firebase Cloud Functions manage the publishing queue and API interactions.
- Integrated Beat Monetization System: Connects to external beat-selling platforms (e.g., BeatStars, Gumroad) to automate the listing of AI-generated beats, manage various licensing options, and enable the setup and promotion of beat pack bundles and sales funnels within YouTube descriptions. Firebase can track sales data.
- Performance Analytics & AI Feedback Loop: Collects and analyzes channel performance data (views, CTR, watch time, beat sales) from YouTube and integrated sales platforms, displayed through a 'Views This Week' chart and key metrics like 'Total Views'. A dedicated AI agent (orchestrated by a Director Agent like OpenClaw) processes this data to identify successful content patterns and inform future content generation strategies, creating a continuous improvement cycle.
- User Approval & Project Monitoring Dashboard: A centralized dashboard (like the 'Lofi factory overview' shown) allowing users to monitor the status of all automated content production jobs, review AI-generated assets before final publication (e.g., 'Halloween Lofi - ready for review'), and provide approval or request revisions, ensuring quality control and user oversight. This includes displaying 'Active Jobs' and 'Recent Jobs' with their current status. Firestore stores job statuses and approval states.

## Style Guidelines:

- The application will utilize a dark color scheme, providing a focused and professional environment for monitoring automated production.
- Background Color: A deep charcoal grey (e.g., #1A1A1A or similar) for main application areas and sidebar, providing a professional and low-distraction backdrop.
- Card/Panel Background: A slightly lighter grey (e.g., #2B2B2B or similar) for content cards, metrics displays (like 'Channels', 'Videos Uploaded'), and data visualizations, creating clear visual separation.
- Text Color: White or very light grey for primary text, ensuring high readability against the dark backgrounds.
- Accent Color: A vibrant, clear blue (e.g., #93ACF0) will highlight interactive elements, active states in navigation, and provide visual contrast, emphasizing actionable insights.
- Status Colors: Green (e.g., #6CC459) for positive metrics or successful actions; Orange/Yellow (e.g., #FFBF00) for items awaiting review or critical attention; Purple (e.g., #A677FF) for data visualization lines (e.g., chart trends); Light grey for 'running now' and 'queued' job statuses.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired feel that resonates with the AI automation and factory aspect.
- Body Font: 'Inter' (sans-serif) for excellent readability and a neutral, professional appearance, suitable for displaying data, statuses, and descriptive text in a high-volume production environment. Different font weights and sizes will differentiate titles, metrics, and explanatory text.
- Utilize minimalist, outline icons (e.g., for dashboard, new project, job queue) that clearly communicate functionality, with a slightly rounded aesthetic to maintain a friendly and modern feel. Status indicators (colored dots) will be used next to job titles.
- A two-column layout featuring a fixed sidebar for navigation (e.g., 'Dashboard', 'Job queue', 'Channels') and a main content area. The main content area, such as the dashboard, will employ a grid-like structure for displaying metrics cards, a chart, and recent job lists, ensuring clear data organization.
- Clear delineation between sections within the sidebar (e.g., 'MAIN', 'ANALYTICS') and a dedicated call-to-action/status bar (e.g., 'Halloween Lofi - ready for review') at the bottom of the main content area for prominent user interaction.
- Incorporate subtle, functional animations for user feedback, such as loading spinners for AI processes, smooth transitions for status updates, and dynamic chart visualizations for performance trends. Progress indicators for automated workflows and dynamic elements will reflect the 'factory's' ongoing output.
